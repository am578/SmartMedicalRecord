# 🚀 Smart Medical App

## 📱 Description

Smart Medical is an Android application designed to manage medical workflows efficiently.  
It connects patients, doctors, and receptionists in one system to handle appointments, patient records, and medical requests.

---

## ✨ Features

- 🔐 Login & Register system  
- 👥 Role-based dashboards:
  - 🧑‍⚕️ Doctor
  - 🧑‍💼 Receptionist
  - 🧑‍🦱 Patient
  - 👑 Admin  

- 📋 Patient management system  
- 🔎 Patients list with search  
- 📅 Appointment request system  
- 🏥 Medical records tracking  

---

## 🧑‍🦱 Patient Features

- ✍️ Submit symptoms  
- 📅 Request appointments  
- 📜 View medical history  
- 👀 Check upcoming appointments  

---

## 🧑‍⚕️ Doctor Features

- 📥 View patient symptoms  
- 💊 Provide initial diagnosis  
- 📁 Access patient records  

---

## 🧑‍💼 Receptionist Features

- 📅 Manage appointment requests  
- 👥 Manage patients list  

---

## 👑 Admin Features

- ⚙️ Manage users  
- 📊 View system statistics  

---

## 🛠️ Tech Stack

- Kotlin 🟣  
- Jetpack Compose 🎨  
- MVVM Architecture 🧠
